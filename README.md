## 🇬🇧 English

An end-to-end data mining project exploring what drives box office revenue — from data cleaning and EDA to machine learning model training and Power BI visualisation.

### ✨ What's Inside

- Data preprocessing pipeline: missing value handling, feature engineering, outlier detection
- Exploratory analysis of how budget, popularity, genres, and ratings correlate with revenue
- ML models: Decision Tree and Logistic Regression for box office profitability classification
- 80% prediction accuracy after model tuning
- Power BI dashboards for presenting findings

### 🛠️ Tech Stack

| Layer | Tools |
|-------|-------|
| Data Processing | Python · pandas · NumPy |
| Visualisation | matplotlib · seaborn · Power BI |
| Machine Learning | scikit-learn (Decision Tree, Logistic Regression) |
| Environment | Jupyter Notebook |

### 📊 Key Findings

- Budget and popularity are the strongest predictors of box office success
- Genre significantly affects profitability variance (action vs. drama)
- High ratings alone do not guarantee box office performance

### 🚀 Getting Started

```bash
git clone https://github.com/amberquu/movie-data-mining-analysis
cd movie-data-mining-analysis
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
jupyter notebook
```

## 🇨🇳 中文

一个完整的数据挖掘项目，从数据清洗、探索性分析到机器学习建模，研究哪些因素影响电影票房收入，最终用 Power BI 呈现分析结论。

### ✨ 项目内容

- 数据预处理流水线：缺失值处理、特征工程、异常值检测
- 探索性分析：预算、热度、类型、评分与票房的相关关系
- 机器学习建模：决策树和逻辑回归，预测电影票房盈利性
- 调参后预测准确率达 80%
- Power BI 可视化仪表盘

### 🛠️ 技术栈

| 模块 | 工具 |
|------|------|
| 数据处理 | Python · pandas · NumPy |
| 可视化 | matplotlib · seaborn · Power BI |
| 机器学习 | scikit-learn（决策树、逻辑回归） |
| 开发环境 | Jupyter Notebook |

### 📊 主要发现

- 预算和热度是预测票房成功的最强特征
- 电影类型对盈利差异影响显著（动作片 vs. 剧情片）
- 高评分不一定等于高票房
